-> Create Thesis Directory.
-> Execute - sencha generate app ALMITOnTheGo ~/Development/dev/Thesis/ALMITOnTheGo from Sencha Touch unzipped directory.
-> Execute - sencha phonegap init com.harvard.thesis.ALMITOnTheGo ALMITOnTheGo from created project folder (/Development/dev/Thesis/ALMITOnTheGo)

RUNNING ON DEVICE
------------------
-> Plugin device and execute - sencha app build -run native from app folder, i.e. ~/Development/dev/Thesis/ALMITOnTheGo

RUNNING ON IOS SIMULATOR
-------------------------
-> Navigate to ~/Development/dev/Thesis/ALMITOnTheGo/phonegap/platforms/ios and open ALMITOnTheGo.xcodeproj with AppCode

-> Any changes to app.scss, run the following command - sencha app build
